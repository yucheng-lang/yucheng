# 语程 (YuCheng) - 中文母语编程语言

> **以语落笔，筑程而生**

## 名称寓意

- **语**：中文语言、编程语言
- **程**：程序、编译进程、编程工程

合起来就是：**以中文为语，编译为程**，一眼就能看懂是做中文编程语言 / 编译器的，直白又高级。

## 项目全套定名

| 项目        | 名称                                |
| ----------- | ----------------------------------- |
| 中文名      | **语程**                            |
| 英文名      | **YuProg**                          |
| 项目缩写    | **YCC**（YuCheng Chinese Compiler） |
| 核心 Slogan | **以语落笔，筑程而生**              |

## 语言特性（v0.4.1）

- **纯中文语法**：40+ 中文关键字，全部中文化
- **缩进推断**：通过缩进自动推断块边界，`结束` 关键字可选
- **动态类型**：数字、字符串、布尔、空、列表、字典、集合、函数
- **内联函数**：`函数(x): 返回 x * 2` 匿名函数表达式，支持作为回调参数
- **40+ 关键字**：涵盖变量声明、条件、循环、函数、结构体、枚举、模式匹配、异常处理、异步/生成器
- **常量保护**：`常量` 声明的值不可重新赋值
- **关键字别名**：`异常`/`捕获`、`在`/`于` 均可互换使用
- **三元运算符**：`条件 ? 真值 : 假值`
- **默认参数**：`函程 f(甲, 乙 = 假)`
- **双变量遍历**：`循环 键, 值 于 字典:`
- **160+ 内置函数**：数学(32)、字符串(20)、数组(20)、日期时间(14)、文件操作(16)、加密编码(11)、类型转换(20+)、系统(9)、JSON(4)、网络(4)、位运算(6)、GUI(83)
- **GUI 组件库**：35 个组件库全部可正常调用，跨平台后端（Windows: Win32，Linux: GTK4，macOS: Cocoa），控件为各平台原生控件
- **编译为独立可执行文件**：`yucheng -c -s 程序.程` 自动内嵌所有依赖库，产出零依赖可执行文件（Windows 约 6 MB，Linux/macOS 约 8 MB）
- **跨平台支持**：Windows / Linux / macOS 三平台运行和编译
- **Rust 实现**：安全、高性能、可扩展

## 代码示例

```语程
// 你好世界
输出 "你好，语程！"

// 变量与运算
设 a = 100
设 b = 3.14
输出 a + b                        // 103.14

// 条件判断
如果 a > 50:
    输出 "a 大于 50"

// 循环
设 i = 0
循环 i < 3:
    输出 i
    i = i + 1

// 函数定义
函程 加(a, b):
    返回 a + b
输出 加(3, 7)                      // 10

// 内联函数
设 平方 = 函数(x): 返回 x * x
输出 平方(5)                        // 25

// 默认参数
函程 问候(姓名, 称呼 = "同学"):
    输出 称呼 + 姓名
问候("张三")                        // 同学张三

// 三元运算符
设 结果 = a > 50 ? "大" : "小"

// 数学函数
输出 绝对值(-42)                   // 42
输出 幂(2, 10)                     // 1024
输出 阶乘(6)                       // 720

// 数组操作
设 数列 = [3, 1, 4, 1, 5, 9]
输出 数组求和(数列)                 // 23
输出 排序(数列)                     // [1, 1, 3, 4, 5, 9]
输出 数组去重(数列)                 // [3, 1, 4, 5, 9]

// 字符串操作
设 文本 = "你好，语程！"
输出 字符串长度(文本)               // 7
输出 包含(文本, "语程")             // 真

// 字典双变量遍历
设 学生 = {"张三": 90, "李四": 85}
循环 名, 分 于 学生:
    输出 名 + ":" + 转文本(分)

// 文件操作
写入文件("test.txt", "你好")
设 内容 = 读取文件("test.txt")

// 加密编码
输出 MD5("你好")                   // MD5 哈希
输出 UUID()                        // 随机 UUID

// JSON
设 数据 = JSON解析('{"名":"张三"}')
输出 数据.名                       // 张三
```

## 编译为独立 EXE

语程支持将 `.程` 源文件编译为独立 Windows EXE：

```语程
:: 独立编译（推荐！自动内嵌所有依赖库）
yucheng -c -s 我的程序.程

:: 指定输出路径
yucheng -c -s 我的程序.程 -o 发布/程序.exe

:: 编译后即时运行
yucheng -c -s -r 我的程序.程

:: 编译为 GUI 程序（无控制台窗口）
yucheng -c -s --gui 我的程序.程

:: 完整组合：独立+GUI+即时运行
yucheng -c -s --gui -r 我的程序.程
```

编译出的 exe 约 6 MB，零 DLL 依赖，可在任意 Windows 电脑直接运行，无需安装任何环境。

交叉编译（在 Windows 上编译 Linux/macOS 程序）：

```bash
yucheng -c -s --target linux Linux示例/01_基础窗口.程
yucheng -c -s --target macos macOS示例/01_基础窗口测试.程
```

在 Linux/macOS 上本机编译：

```bash
# Linux（需先安装 GTK4 开发库）
sudo apt install libgtk-4-dev          # Ubuntu/Debian
cargo build --release --features gui

# macOS（Cocoa 框架已内置）
cargo build --release --features gui
```

```语程
:: 编译 GUI 程序
yucheng -c -s --gui GUI窗口_登录界面示例.程
:: 产出：GUI窗口_登录界面示例.exe，双击即可弹出窗口（无控制台黑框）
```
