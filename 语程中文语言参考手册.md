# 语程语言参考手册

> 版本：v0.4.1 | 更新日期：2026年5月16日（第4次修订）

## 1. 语言概述

语程（YuCheng / YuProg）是一门面向中文用户的动态类型编程语言。其设计理念是让中文母语者能够用最自然的中文表达方式编写程序，降低编程学习门槛。

> **以语落笔，筑程而生**

| 项目       | 说明                             |
| ---------- | -------------------------------- |
| 语言名称   | 语程（YuProg）                   |
| 当前版本   | v0.4.1                           |
| 源文件后缀 | `.程`                            |
| 运行平台   | Windows 7+ 64 位 / Linux / macOS |
| 实现语言   | Rust 2021 Edition                |

### 1.1 设计原则

- **母语优先**：所有关键字、运算符、内置函数均为中文
- **简洁明了**：语法设计尽量精简，避免冗余符号
- **渐进学习**：从简单到复杂，初学者可以快速上手
- **功能完备**：数学、字符串、数组、文件、加密、JSON、网络等内置函数

### 1.2 文件后缀

语程源文件使用 `.程` 后缀。

### 1.3 执行方式

语程当前为解释执行模式，由 Rust 实现的解释器直接运行 AST。支持以下运行方式：

```
yucheng <文件名.程>          # 运行源文件
yucheng --repl / yucheng -i # 交互式 REPL 模式
yucheng --eval "代码" / -e   # 执行单行代码
yucheng --lex <文件名.程>    # 仅词法分析
yucheng --parse <文件名.程>  # 仅语法分析
yucheng -c -s <文件名.程>    # 编译为独立可执行文件（内嵌依赖库）
yucheng -c -s <文件名.程> -o 输出.exe  # 指定输出路径
yucheng -c -s -r <文件名.程> # 编译并即时运行
yucheng -c -s --gui <文件名.程>  # 编译为 GUI 程序（无控制台窗口）
yucheng -c -s --gui -r <文件名.程>  # 完整组合
yucheng -c -s --target linux <文件名.程>  # 交叉编译 Linux 程序
yucheng -c -s --target macos <文件名.程>  # 交叉编译 macOS 程序
```

不带参数启动时自动进入 REPL 模式。

### 1.4 编译为独立可执行文件

语程支持将 `.程` 源文件编译为独立的可执行文件：

```bash
# 独立编译（推荐！自动内嵌所有依赖库）
yucheng -c -s 程序.程

# 指定输出路径
yucheng -c -s 程序.程 -o 发布/程序.exe

# 编译后即时运行
yucheng -c -s -r 程序.程

# 编译为 GUI 程序（无控制台窗口）
yucheng -c -s --gui 程序.程

# 交叉编译 Linux 程序
yucheng -c -s --target linux 程序.程

# 交叉编译 macOS 程序
yucheng -c -s --target macos 程序.程

# 完整组合
yucheng -c -s --gui -r 程序.程 -o 输出.exe
```

编译出的 Windows exe 约 6 MB，零 DLL 依赖，可在任意 Windows 电脑直接运行。Linux/macOS 可执行文件约 8 MB。

**参数说明**：

| 参数 | 短参数 | 说明 |
|------|--------|------|
| `--compile` | `-c` | 启用编译模式 |
| `--standalone` | `-s` | 自动内嵌所有依赖库 |
| `--output 路径` | `-o` | 指定输出路径 |
| `--run` | `-r` | 编译后即时运行（交叉编译时无效） |
| `--gui` | — | 编译为 GUI 程序（无控制台窗口，仅 Windows 目标有效） |
| `--target 平台` | — | 指定目标平台：windows / linux / macos |

**原理**：采用"壳程序"技术——将源码和所有依赖库追加到 stub 可执行文件末尾。运行时自动检测附加数据，直接解释执行内嵌源码。使用 `-s` 参数会自动递归收集所有 `导入` 语句对应的库文件并内嵌到可执行文件中，实现完全独立运行。`--gui` 参数修改 PE 头子系统为 WINDOWS（而非 CONSOLE），使双击运行时不弹出控制台黑框窗口。`--target` 参数指定目标平台，使用对应平台的 stub 文件实现交叉编译。

| 类型 | 说明 | 示例 |
|------|------|------|
| 数字 | 整数和浮点数 | `42`, `3.14`, `-7` |

## 4. 运算符

### 4.1 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法/字符串连接 | `3 + 4` → `7` |
| `-` | 减法 | `10 - 3` → `7` |
| `*` | 乘法 | `3 * 4` → `12` |
| `/` | 除法 | `10 / 3` → `3.333...` |
| `%` | 取模 | `10 % 3` → `1` |

### 4.2 比较运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `==` | 等于 | `3 == 3` → `真` |
| `!=` | 不等于 | `3 != 4` → `真` |
| `<` | 小于 | `3 < 4` → `真` |
| `>` | 大于 | `4 > 3` → `真` |
| `<=` | 小于等于 | `3 <= 3` → `真` |
| `>=` | 大于等于 | `4 >= 3` → `真` |

> 比较运算符同时支持数字和字符串。字符串按字典序比较：`"abc" < "def"` → `真`

### 4.3 逻辑运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `且` | 逻辑与 | `真 且 假` → `假` |
| `或` | 逻辑或 | `真 或 假` → `真` |
| `非` | 逻辑非 | `非 真` → `假` |

### 4.4 位运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `&` | 位与 | `5 & 3` → `1` |
| `\|` | 位或 | `5 \| 3` → `7` |
| `^` | 位异或 | `5 ^ 3` → `6` |
| `~` | 位取反 | `~5` → `-6` |
| `<<` | 左移 | `1 << 3` → `8` |
| `>>` | 右移 | `8 >> 2` → `2` |

### 4.5 三元运算符

```
条件 ? 真值 : 假值
```

示例：`年龄 >= 18 ? "成年" : "未成年"`

## 5. 控制流

### 5.1 条件语句

```
如果 条件:
    // 条件为真时执行
否则如果 另一条件:
    // 另一条件为真时执行
否则:
    // 所有条件为假时执行
```

> `结束` 关键字为可选。语程通过缩进自动推断块边界。

### 5.2 循环（while）

```
循环 条件:
    // 条件为真时重复执行
```

### 5.3 遍历循环（for）

```
遍 变量 于 列表:
    // 对每个元素执行
```

也支持遍历字符串：

```
遍 字 于 "你好":
    输出 字
```

遍历字典的键：

```
遍 键 于 字典:
    输出 键
```

双变量遍历字典（同时获取键和值）：

```
遍 键, 值 于 字典:
    输出 键 + " = " + 转文本(值)
```

### 5.4 循环控制

- `跳出`：跳出当前循环
- `继续`：跳过本次迭代

## 6. 函数

### 6.1 定义函数

```
函程 函数名(参数1, 参数2):
    // 函数体
    返回 返回值
```

> **注意**：语程通过缩进自动推断块边界，`结束` 关键字为可选。`返回` 是返回值关键字（对应 return）。

### 6.2 调用函数

```
设 结果 = 函数名(参数1, 参数2)
```

### 6.3 默认参数

语程支持函数默认参数：

```
函程 问候(名字, 称呼 = "先生"):
    输出 "你好，" + 称呼 + 名字
结束

问候("张三")           // 你好，先生张三
问候("张三", "女士")   // 你好，女士张三
```

### 6.4 匿名函数（内联函数）

使用 `函数` 关键字定义匿名函数表达式：

```
// 内联函数赋值给变量
设 加法 = 函数(a, b): 返回 a + b
输出 加法(3, 4)  // 7

// 作为回调参数
设 结果 = 映射([1, 2, 3], 函数(x): 返回 x * x)
输出 结果  // [1, 4, 9]

// 带默认参数的内联函数
设 问候 = 函数(名, 语气="你好"): 返回 语气 + "，" + 名
输出 问候("世界")     // 你好，世界
输出 问候("语程", "嗨")  // 嗨，语程

// 多行内联函数（字典方法）
设 对象 = {
    计算(参数):
        设 结果 = 参数 * 2
        返回 结果
}
```

### 6.3 递归

语程完全支持递归调用：

```
函程 斐波那契(n):
    如果 n <= 1:
        返回 n
    返回 斐波那契(n - 1) + 斐波那契(n - 2)
```

## 7. 列表

### 7.1 创建列表

```
设 数列 = [1, 2, 3, 4, 5]
设 混合 = [1, "二", 真, 空]
```

### 7.2 访问元素

```
输出 数列[0]    // 1
输出 数列[4]    // 5
```

### 7.3 列表操作函数

```
输出 数组长度(数列)    // 5
输出 排序(数列)        // [1, 2, 3, 4, 5]
输出 反转(数列)        // [5, 4, 3, 2, 1]
输出 数组求和(数列)    // 15
输出 数组去重(数列)    // 去除重复
输出 数组洗牌(数列)    // 随机打乱
```

### 7.4 列表方法

列表对象支持以下方法调用：

```
数列.长度          // 获取长度
数列.添加(值)      // 添加元素
数列.删除(索引)    // 删除元素
数列.包含(值)      // 是否包含
数列.排序          // 排序
数列.反转          // 反转
```

#### 索引访问

列表和字符串支持索引访问，包括**负数索引**：

```
设 列表 = [10, 20, 30, 40, 50]
列表[0]    // 10（第一个元素）
列表[-1]   // 50（最后一个元素）
列表[-2]   // 40（倒数第二个元素）

设 文本 = "你好世界"
文本[0]    // "你"
文本[-1]   // "界"
```

## 14.1 REPL 交互模式

语程支持交互式 REPL（Read-Eval-Print Loop）模式，适合快速测试和实验。

### 启动方式

```
yucheng           # 不带参数，自动进入 REPL
yucheng --repl    # 显式指定 REPL 模式
yucheng -i        # 简写
```

### REPL 命令

| 命令 | 说明 |
|------|------|
| `退出` / `exit` / `quit` | 退出 REPL |
| `清空` / `clear` | 清空输入缓冲区 |
| `帮助` / `help` | 显示帮助信息 |

### REPL 示例

```
语程> 设 x = 42
语程> 输出 x * 2
84
语程> 函程 问候(名字): 输出 "你好，" + 名字 结束
语程> 问候("语程")
你好，语程
语程> 退出
再见！
```

### 多行输入

当输入的代码不完整（如缺少 `结束`）时，REPL 会自动进入续行模式：

```
语程> 函程 加(a, b):
...     返回 a + b
... 结束
语程> 加(3, 4)
7
```
