# 语程语言 VSCode 插件安装说明

## 插件文件位置
`D:\语程语言\vscode插件\yucheng-syntax\yucheng-syntax-0.4.1.vsix`

## 安装方法

### 方法一：通过 VSCode 命令面板安装（推荐）

1. 打开 Visual Studio Code
2. 按 `Ctrl + Shift + P` 打开命令面板
3. 输入 `Extensions: Install from VSIX...` 并选择
4. 浏览到插件文件位置：`D:\语程语言\vscode插件\yucheng-syntax\yucheng-syntax-0.4.1.vsix`
5. 选择文件并安装
6. **重启 VSCode**

### 方法二：通过命令行安装

```powershell
code --install-extension "D:\语程语言\vscode插件\yucheng-syntax\yucheng-syntax-0.4.1.vsix"
```

## 配置语程解释器

### 方法一：配置环境变量（推荐）

将 yucheng.exe 所在目录添加到系统环境变量 PATH 中：

1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
2. 在"系统变量"中找到 `Path`，点击编辑
3. 添加 yucheng.exe 所在目录，例如：
   ```
   D:\语程语言\target\release
   ```
4. 确定保存，重启 VSCode

配置后，插件会自动使用 `yucheng` 命令运行文件。

### 方法二：在 VSCode 中设置路径

如果不想配置环境变量，可以在 VSCode 设置中指定完整路径：

1. 按 `Ctrl + ,` 打开设置
2. 搜索 "语程"
3. 修改 "Yucheng Path" 为完整路径，例如：
   ```
   D:\语程语言\target\release\yucheng.exe
   ```

## 使用方法

### 运行语程文件
1. 打开 `.程` 文件
2. 按 `F6` 运行
3. 或右键选择 "运行当前文件"

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `F6` | 运行当前文件 |
| `Shift+F5` | 调试当前文件 |
| `Ctrl+Alt+B` | 检查代码 |
| `Shift+Enter` | 在终端中运行选中内容 |

### 代码片段
输入关键字前缀 + `Tab` 快速生成代码：
- `函程` → 函数定义
- `如果` → 条件判断
- `循环` → 条件循环
- `遍` → 遍历循环
- `尝试` → 异常处理

## 问题排查

### 如果运行时提示找不到 yucheng

**方法一：检查环境变量**
```powershell
# 在终端中测试
yucheng --version
```
如果提示找不到命令，说明环境变量未配置正确。

**方法二：在 VSCode 设置中指定路径**
1. 按 `Ctrl + ,` 打开设置
2. 搜索 "yucheng.yuchengPath"
3. 设置为完整路径，如 `D:\语程语言\target\release\yucheng.exe`

### 如果 PowerShell 报错
插件已修复 PowerShell 命令格式问题，会自动使用正确的调用方式：
- 简单命令：`yucheng "file.程"`
- 完整路径：`& "D:\path\yucheng.exe" "file.程"`
